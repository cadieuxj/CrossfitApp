The user is developing ApexAI Athletics, an Android app that combines:
- CrossFit workout tracking (WODs, PRs, timers)
- AI-powered video movement coaching via Google Gemini 1.5 Pro
- Athlete readiness scoring using ACWR (Acute:Chronic Workload Ratio)
- Health Connect biometric integration (HRV, sleep, resting HR)

Stack: Android (Kotlin/Compose), FastAPI (Python) backend, Supabase (PostgreSQL), Google Gemini API.
Jurisdiction: Quebec, Canada — Law 25 (Loi 25) applies.
Status as of 2026-03-28: pre-production, single initial commit, not yet deployed.
